## [0.1.0] - 2024-01-15

### Added
- Initial release of TextCraft
- Word Counter tool with real-time analysis
- Case Converter tool with multiple case types
- Line Breaks Remover tool for text cleanup
- Text Diff Checker for comparing texts
- Lorem Ipsum Generator for placeholder text
- Responsive design for mobile and desktop
- Dark/light theme support
- Progressive Web App (PWA) features
- TypeScript support throughout
- Comprehensive test suite
- Documentation generation with TypeDoc
- ESLint and Prettier configuration
- Vite build system for fast development

### Features
- **Word Counter**: Count words, characters, sentences, paragraphs with reading time estimates
- **Case Converter**: Transform text between UPPERCASE, lowercase, Title Case, camelCase, kebab-case, snake_case
- **Line Breaks Remover**: Clean up whitespace and normalize text formatting
- **Text Diff Checker**: Compare two texts with side-by-side highlighting
- **Lorem Ipsum Generator**: Generate placeholder text of various types and lengths

### Technical
- Built with React 18 and TypeScript
- Styled with Tailwind CSS
- UI components from Radix UI
- Icons from Lucide React
- Testing with Vitest and Testing Library
- Documentation with TypeDoc
- Build system with Vite
- PWA support with service worker
