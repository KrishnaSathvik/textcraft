## 🛠️ Tools Included

### 1. **Word Counter** (`/word-counter`)
- Count words, characters, sentences, paragraphs
- Reading time estimates (reading and speaking)
- Character density analysis
- Copy and download functionality
- Real-time statistics

### 2. **Case Converter** (`/case-converter`)
- Transform text between different cases
- Supported: UPPERCASE, lowercase, Title Case, camelCase, kebab-case, snake_case
- Real-time preview
- Undo functionality
- Apply changes with one click

### 3. **Line Breaks Remover** (`/line-breaks`)
- Clean up and normalize text whitespace
- Remove extra spaces and line breaks
- Batch processing capabilities
- Export cleaned text

### 4. **Text Diff Checker** (`/diff-checker`)
- Compare two texts side by side
- Highlight differences visually
- Show statistics (added, removed, unchanged)
- Export diff results

### 5. **Lorem Ipsum Generator** (`/lorem-ipsum`)
- Generate placeholder text
- Multiple formats (paragraphs, sentences, words)
- HTML and Markdown support
- Copy and download functionality
